STUDENT GRADE EVALUATION

A simple web app that calculates a student's average score from 4 subjects and checks if they Pass or Fail.

 Features
- Input grades for MATH, ENGLISH, PHYSICS, and CHEMISTRY
- Validates that scores are between 0 and 100
- Calculates the average: total / 4
- Shows result: `Pass` if average >= 50, else `Fail`
- Clean, interactive UI with color feedback

How to Use
- Open `index.html` in your browser
- Enter grades for all 4 subjects
- Click `SUBMIT` to see your result


Tech Used-
HTML, CSS, JavaScript
